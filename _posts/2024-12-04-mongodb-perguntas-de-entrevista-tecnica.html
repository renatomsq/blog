Segue abaixo uma relação de perguntas relacionadas a MongoDB que eu já presenciei em algumas entrevistas. Não é uma lista exaustiva apesar do tamanho (a única coisa gerada por IA neste post é a imagem mesmo) mas cobre bastante do que é perguntado.

Lembrando, são perguntas genéricas partindo de pontos técnicos. Sempre dá pra apimentar e ir mais deep em assuntos de interesse (arquiteturais por exemplo) e a maior parte do tempo são perguntas do que você sabe e o que você fez com Mongo, que são perguntas mais abertas e que dizem muito sobre os candidatos. Sem mais delongas:

  1. O que é NoSQL?

  2. Qual diferença do MongoDB para outras soluções NoSQL?

  3. Qual diferença do MongoDB para outras soluções de bancos relacionais?

  4. Como funciona a Replicação do MongoDB? (Leia-se Replicaset)

  5. Como funciona o Particionamento do MongoDB? (Leia-se Sharding)

  6. Como você faz o MongoDB escalar verticalmente?

  7. Como você faz o MongoDB escalar horizontalmente?

  8. Para que serve o Mongod.conf ?

  9. Como você monitora o MongoDB do seu ambiente?

  10. Como você faz backup do MongoDB do seu ambiente?

  11. Como funciona um índice no MongoDB e quais tipos?

  12. Como funciona o Write-Concern?

  13. O MongoDB suporta transações?

  14. O MongoDB suporta transações multi-documento?

  15. Como faço join entre collections diferentes no MongoDB?

  16. Qual a diferença entre mongos, mongod e mongoshell.

  17. Quais opções de backup existe no MongoDB?

  18. O que é oplog?

  19. O que é sharding e para que serve?

  20. Explique a topologia de um Sharding

  21. O que é uma shark Key

  22. Como você aplicaria atualizações de versão no MongoDB?

  23. O que é documento e coleção no MongoDB?

  24. (Conte uma história de algum problema real) -Você usaria MongoDB para resolver esse problema ou usaria outra solução?

  25. O que é WiredTiger?

  26. O que é MongoDB Compass?

  27. O que é MongoDB Ops Manager?

  28. O que é journaling?

  29. O que é readConcern?

  30. Como funciona a autenticação e autorização no Mongodb?

  31. O que é MongoDB Atlas?

  32. Porque usar Atlas e não MongoDB em alguma EC2/VM?

Fique à vontade para acrescentar alguma questão se quiser ou compartilhar algo sobre este assunto.
